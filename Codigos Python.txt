python -m venv modules ---> se crea para tener un entorno virtual, y poder instalar
	los paquetes o dependencias de python dentro del proyecto --> pip install pandas

pip freeze -->para ver listado de paquetes instalados

pip freeze > paquetes.txt  --->Forma de actualizar el archivo de paquetes.
